## wan_2p5 url
支持官方api
北京地域官方url，dashscope.base_http_api_url = 'https://dashscope.aliyuncs.com/api/v1'
若使用新加坡地域的模型，需将url替换为：https://dashscope-intl.aliyuncs.com/api/v1
## sora2 url
支持官方api，https://api.openai.com/v1，同时支持三方url
## veo3 url
暂时仅支持三方url，如https://api.newcoin.top，格式为openai chat completion格式，需注意不一定中转所有参数，response下载路径暂时没有实现因为感觉各种三方转接的response不太一样
